
# Gold Price Trend Analysis

## Overview
This project analyzes historical gold price data to identify long-term trends,
volatility patterns, and relationships between price variables.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Analysis Performed
- Exploratory Data Analysis (EDA)
- Year-wise trend analysis
- Moving averages (10-week and 50-week)
- Volatility and weekly returns analysis
- Correlation analysis using heatmaps

## Key Insights
- Gold prices show a long-term upward trend over the years
- Strong correlation exists between Open, High, Low, and Close prices
- Volatility increases during uncertain economic periods
